How VCI & NDVI Monitor Drought

NDVI (Normalized Difference Vegetation Index) measures green vegetation vigor using satellite reflectance. It scales from -1 to +1, where dense green vegetation falls above 0.5 and water or soil has negative/near-zero values. However, absolute NDVI cannot measure anomalies: a desert always has low NDVI but is not necessarily in "drought."

VCI (Vegetation Condition Index) standardizes NDVI by comparing the current NDVI value to its historical minimum and maximum for that exact pixel and time of year. VCI scales from 0 to 100:

  • VCI = 100: Best vegetation condition recorded in history.
  • VCI = 0: Worst vegetation condition recorded in history.
  • VCI ≤ 35: Signifies severe-to-extreme agricultural drought.
Mathematical Formula
\[NDVI = \frac{\text{NIR} - \text{Red}}{\text{NIR} + \text{Red}}\]
\[VCI = \frac{NDVI_{\text{current}} - NDVI_{\text{min}}}{NDVI_{\text{max}} - NDVI_{\text{min}}} \times 100\]

Where $NDVI_{\text{min}}$ and $NDVI_{\text{max}}$ are pixel-by-pixel multi-year envelope parameters (2015-2024).

scripts/calculate_indices.py 
import ee
import config

# Initialize Earth Engine
ee.Initialize()

# Calculate pixel-wise min and max NDVI historically
modis = ee.ImageCollection("MODIS/061/MOD13Q1") \
          .filterBounds(roi) \
          .filter(ee.Filter.calendarRange(2015, 2024, 'year')) \
          .filter(ee.Filter.calendarRange(8, 8, 'month')) \
          .select('NDVI')

scale_modis = lambda img: img.multiply(0.0001)
modis_scaled = modis.map(scale_modis)

ndvi_min = modis_scaled.min()
ndvi_max = modis_scaled.max()

# Load current Sentinel-2 imagery
s2_current = ee.ImageCollection("COPERNICUS/S2_SR_HARMONIZED") \
               .filterBounds(roi) \
               .filterDate('2025-08-01', '2025-09-01') \
               .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 20)) \
               .median()

# Compute current NDVI
ndvi_current = s2_current.normalizedDifference(['B8', 'B4'])

# Calculate Vegetation Condition Index (VCI)
vci = ndvi_current.subtract(ndvi_min) \
                  .divide(ndvi_max.subtract(ndvi_min)) \
                  .multiply(100.0) \
                  .clamp(0, 100)
